#### 2. 语法分析
- 很常规，给一个文法，证明是LR(1)文法但不是LALR(1)文法
增广文法，画LR(1)自动机，找同心项集，举例说明合并后有规约-规约冲突即可
- 还要证明改造后的文法不是LL(1)文法
提左公因子，消除左递归，FIRST集，FOLLOW集，SELECT集，说明同一左部的产生式的SELECT集有相交
