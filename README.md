# 大三下课程资料总览

本目录整理 2025-2026 学年春季学期的大三下课程资料。每个课程文件夹下都有单独的 `README.md`，用于说明材料内容、课程成绩、授课老师和一些主观杂谈。

## 大三下需要做的

大三下成绩相对没那么重要，对于未来技术的学生来说，保证保研资格为首要，同时无论是校内保研还是保外都要进入面试/笔试阶段，建议准备好简历和个人陈述PPT，保内导师可以参考HITCS的文档、在校园集市冲浪或咨询学长

## 选课建议

选修课有4分要求的同时，未来技术有10分选修要求，选课前可以查看课程大纲看是否有考试，原则上要不有用/要不省事，自认为我选的这几门课都可以算有用或省事.
