# 大模型技术与原理

## 材料说明

- `PPT/`：课程主课件，覆盖语言模型发展、Transformer、模型配置、长上下文、预训练数据与训练优化、提示工程、上下文学习、思维链、指令微调、轻量化微调、人类对齐和大模型评测。
- `LLM_lab/`：实验资料与个人实验产物，包含 LAB1-LAB4，能看到实验报告、代码、配置、数据、训练输出、评测结果和提交包。
- `LAB/`：课程实验相关补充目录。
