# （2）实现 DiskManager::create_file 函数

函数声明： 

```cpp
void DiskManager::create_file(const std::string &path); 
```

功能：创建⽂件。参数的含义参考代码注释。 

实现：参考代码注释。基本实现⽅法是调⽤open函数，使⽤O_CREAT模式。注意不能重复创建相同⽂件，否则抛 出异常（阅读 src/test/storage/disk_manager_test.cpp ⽂件，了解这种情况下抛出异常的类型）。 

# （3）实现 DiskManager::destroy_file 函数

函数声明： 

```cpp
void DiskManager::destroy_file(const std::string &path); 
```

功能：删除⽂件。参数的含义参考代码注释。 

实现：参考代码注释。基本实现⽅法是调⽤unlink函数。注意不能删除未关闭的⽂件，否则抛出异常（阅读 src/test/storage/disk_manager_test.cpp ⽂件，了解这种情况下抛出异常的类型）。 

# （4）实现 DiskManager::open_file 函数

函数声明： 

```cpp
int DiskManager::open_file(const std::string &path); 
```

功能：打开⽂件。参数和返回值的含义参考代码注释。 

实现：参考代码注释。基本实现⽅法是调⽤open函数，使⽤O_RDWR模式，然后更新打开⽂件的列表， 即path2fd_和fd2path_。注意不能重复打开相同⽂件，否则抛出异常（阅读 src/test/storage/disk_manager_test.cpp ⽂件，了解这种情况下抛出异常的类型）。 

# （5）实现 DiskManager::close_file 函数

函数声明： 

```txt
void DiskManager::close_file(int fd); 
```

功能：关闭⽂件。参数的含义参考代码注释。 

实现：参考代码注释。基本实现⽅法是调⽤ close 函数，然后更新打开⽂件的列表，即 path2fd_ 和 fd2path_ 。 注意不能重复打开相同⽂件，否则抛出异常（阅读 src/test/storage/disk_manager_test.cpp ⽂件，了解这 种情况下抛出异常的类型）。 

# （6）实现 DiskManager::read_page 函数

函数声明： 

```cpp
void DiskManager::read_page(int fd, page_id_t page_no, char *offset, int num_bytes); 
```

功能：读⻚⾯。参数的含义参考代码注释。 

实现：参考代码注释。基本实现逻辑如下： 

1. 根据 page_no 和 PAGE_SIZE 计算指定⻚⾯在⽂件中的偏移量。 

2. 调⽤ read 函数。 

⼀般情况下， num_bytes = PAGE_SIZE 。但也可以是 num_bytes < PAGE_SIZE ，如只读取⻚头。 

# （7）实现 DiskManager::write_page 函数

```cpp
void DiskManager::write_page(int fd, page_id_t page_no, const char *offset, int num_bytes); 
```

功能：写⻚⾯。参数的含义参考代码注释。 

实现：参考代码注释。基本实现逻辑如下： 

1. 根据 page_no 和 PAGE_SIZE 计算指定⻚⾯在⽂件中的偏移量。 

2. 调⽤ write 函数。 

⼀般情况下， num_bytes = PAGE_SIZE 。但也可以是 num_bytes < PAGE_SIZE ，如只读取⻚头。
