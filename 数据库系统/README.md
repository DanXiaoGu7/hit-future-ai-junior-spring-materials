# 数据库系统

## 材料说明

- `PPT/`：课程主课件，包含 ch1-ch11 handout。
- `quiz/`：章节 quiz，覆盖数据库基础、关系模型、索引、存储介质等内容。
- `homework/`：章节作业与 `作业分数表.csv`。
- `DatabaseSystemLAB/`：数据库系统实验目录，包含 LAB0-LAB4、实验报告、验收复习文档和 `rucbase-lab` 代码。
- `HIT-database_systems_2025/`：历史资料包，包含课件、实验、往年题、作业答案和开箱手册。
- `成绩构成.png`：课程考核方式截图。
